### 2024.4.18 回文数  https://leetcode.cn/problems/palindrome-number/
---
### JS
```JS
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = function(x) {
    if(x < 0 || (!(x % 10) && x)) return false;
    let xTmp = x
    let res = 0
    while(xTmp > 0) {
        res = res * 10 + xTmp % 10
        xTmp = Math.floor(xTmp / 10)
    }
    return x === res
};
```
注意：负数 10的倍数都不是回文数字

**时间复杂度：** 循环执行的次数与入参的位数有关，`O(n)`

**空间复杂度：** 只存储xTmp和res两个变量，`O(1)`
